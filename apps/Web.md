# Model-View-Controller (MVC)

## Base

```html
<html>
  <head>
    <!-- Head -->
  </head>
  <body>
    <!-- Body -->
  </body>
</html>
```

## Application (Model)

- List: Default-View
- Type: Return-Types
- Form: Inputs-Types

```html
<main>
  <!-- @Model-View -->
  <section>
    <!-- LIST -->
    <table></table>
    <!-- TYPE -->
    <article>Database Model</article>
    <!-- FORM -->
    <form></form>
  </section>
</main>
```
